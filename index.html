Countdown to Isles Hockey!

This website counts down the number of days until the friggin puck drops for the 2017-1028 season. When the amount of days remaining matches a roster players sweater number, you'll get a treat. Otherwise, you'll get a random image for a sweet background. Reload for new stuff.

This site was made by Chad Moore, on the seacoast of New Hampshire (I grew up on Long Island). The code is on Github.
